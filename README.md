
# Brick n Balls

Small game made in unity 6000.0.63f1 which uses ECS only for physics (rest is handled in standard Gameobjects)




## Installation and usage 

Download source and go to Unity Hub -> Add -> Add project from disk. 

Starting scene is "BootScene" (Build index 0). Use it to properly initalize all game systems.
